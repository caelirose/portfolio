Sales Performance Dashboard

Designed and built an end-to-end sales performance dashboard for a retail dataset. The project involved cleaning raw transactional data using Python (Pandas), loading it into a PostgreSQL database with optimized SQL queries, and visualizing key KPIs in Tableau — including revenue trends, regional breakdowns, and product-level performance.

The dashboard provided stakeholders with actionable insights that helped identify underperforming regions and reallocate marketing budget, resulting in a projected 15% improvement in quarterly sales targeting accuracy.
